# CasaSalud.cl Website

A modern, responsive website for CasaSalud.cl - a platform focused on integral health and well-being.

## Overview

This website recreates the beautiful CasaSalud.cl design with a focus on:
- **Integral Health**: Comprehensive approach to physical, mental, and emotional well-being
- **Multiprofessional Team**: Diverse team of health professionals
- **Online Consultations**: Virtual healthcare services
- **Community Focus**: Building a supportive health community

## Features

### Design Elements
- **Green Watercolor Theme**: Calm, natural aesthetic with various shades of green
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Smooth Animations**: Subtle animations and transitions for better user experience
- **Modern Typography**: Clean, readable Inter font family

### Sections
1. **Hero Section**: Logo, tagline, and navigation buttons
2. **Multiprofessional Services**: Team approach to healthcare
3. **Service Showcase**: Three main service categories
4. **Booking Section**: Easy appointment scheduling
5. **Patient Testimonials**: Real patient feedback
6. **Join the Team**: Contact information for professionals
7. **About Catalina**: Founder's profile and qualifications
8. **Platform Information**: Features for healthcare professionals
9. **Vision & Mission**: Company values and goals

### Interactive Features
- **Smooth Scrolling**: Navigation buttons scroll to relevant sections
- **Booking Modal**: Interactive appointment booking interface
- **Contact Integration**: Click-to-copy phone and email functionality
- **Hover Effects**: Enhanced user interaction with cards and buttons
- **Scroll Animations**: Elements animate into view as you scroll

## Technology Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with Tailwind CSS framework
- **JavaScript**: Interactive functionality and animations
- **Responsive Design**: Mobile-first approach

## Getting Started

1. **Open the website**: Simply open `index.html` in your web browser
2. **Local server** (recommended): Use a local server for best experience
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

## Customization

### Colors
The website uses a green color palette defined in CSS variables:
- Primary Green: `#16a34a`
- Light Green: `#dcfce7`
- Dark Green: `#15803d`
- Watercolor Green: `#bbf7d0`

### Content
- Update contact information in the HTML
- Replace emoji placeholders with actual images
- Modify text content to match your needs
- Add your own branding elements

### Styling
- Modify `styles.css` for custom styling
- Adjust Tailwind classes in HTML for quick changes
- Update color variables for different themes
